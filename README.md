## Usage Plan:

Each developer checks in code at the end of each working session. Code files are stored in a dedicated folder structure, including source code (src/), UML diagrams (uml/), and documentation (docs/).

## Layout:

- src: code files
- uml: UML diagram
- doc: documents
- tests: test cases
